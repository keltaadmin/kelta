# Member Card Subsystem

## Purpose

Responsible for building digital member cards.

This subsystem does **not**:

- Issue digital identities
- Generate QR codes
- Apply business rules

Instead, it assembles information from:

- Member
- Digital Identity
- Platform QR Infrastructure

into a presentation-ready card.

## Consumers

- Mobile App
- Web Portal
- Printable Card
- PDF Generator
- Future Wallet Integration
